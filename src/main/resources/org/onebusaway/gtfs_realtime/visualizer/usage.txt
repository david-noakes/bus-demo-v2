Description:
  Visualize GTFS-realtime vehicle position data.
  
Usage:
  java -jar demo.jar --vehiclePositionsUrl=url [--port=8080] [--TracksUpdate] [--TracksSimulate]


Args:
  --vehiclePositionsUrl=url           GTFS-realtime vehicle positions url
  --port=8080                         Optional. port number to use (default = 8080)
  --TracksUpdate                      Optional - if present, update Google Map Tracks with information from the realtime feed
  --TracksSimulate                    Optional - if present, do not call Google Tracks, but load responses from files 
  
Examples:
Translink SEQ feed     --vehiclePositionsUrl=https://gtfsrt.api.translink.com.au/Feed/SEQ
Sydney Transport       --vehiclePositionsUrl=http://dev.triptasticapp.com/au_sydney/tools/GTFSRSydneyAlerts
MTA                    --vehiclePositionsUrl=http://developer.mbta.com/lib/gtrtfs/Vehicles.pb
